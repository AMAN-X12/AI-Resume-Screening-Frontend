# 📄 AI Resume ATS Screener (Frontend)

It is the frontend interface for the AI Resume Screener project! This application provides a clean, user-friendly UI for uploading resumes, inputting required skills, and utilizing AI to extract and match key candidate information.

**Live Demo:** [https://ai-resume-screening-frontend-psi.vercel.app](https://ai-resume-screening-frontend-psi.vercel.app)

---

## ⚙️ System Architecture

This project is built using a decoupled, full-stack architecture. This repository contains **only the client-side user interface(Frontend)**.

The actual AI processing (NLP/Spacy), file parsing, and MongoDB database connections are handled by a dedicated Python server.To Get the backend of this project,kidly do visit :

* **Backend Repository:** ['https://github.com/AMAN-X12/AI-Resume-ATS-Backend'](https://github.com/AMAN-X12/AI-Resume-ATS-Backend)

* **Live API Server:** Hosted on Hugging Face

##  Built With

* **HTML5 / CSS3** for structure and styling.
* **Vanilla JavaScript** for asynchronous API communication (`fetch`).

* **Vercel** Deployment of this client side user interface(frontend) for hosting purpose.
